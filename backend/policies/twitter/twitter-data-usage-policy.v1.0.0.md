# Twitter Data Usage Policy

**Version:** 1.0.0  
**Last Updated:** 2026-02-02

---

## 1. Overview

This policy explains how we collect, process, and use Twitter-related data when you connect your Twitter account to our platform.

By accepting this policy, you explicitly consent to the data practices described below.

---

## 2. What Data We Collect

When you connect Twitter via our Chrome Extension, we may collect:

- Authentication cookies required to establish a Twitter session
- Public account metadata (username, user ID)
- Session technical metadata (timestamps, browser type, domain)

**⚠️ We do NOT collect:**
- Twitter passwords
- Direct Messages (DMs)
- Private account content
- Email addresses
- Payment or billing information

---

## 3. How Data Is Collected

Data is collected **only** when:
- You install the official Chrome Extension
- You explicitly click **"Sync"** inside the extension

We do not perform background or silent data collection.

---

## 4. Purpose of Data Usage

Collected data is used exclusively for:

- Establishing and maintaining an authenticated Twitter session
- Executing user-initiated analytics, monitoring, or parsing tasks
- Detecting session health and preventing abuse or misuse
- Generating crypto signals and sentiment analysis from public tweets

---

## 5. Data Storage & Security

- Cookies are encrypted at rest using AES-256-GCM
- Access is restricted to internal processing services
- Data is never shared with third parties
- Sessions can be revoked at any time by the user
- All data is stored on encrypted volumes

---

## 6. User Control & Revocation

You may:
- Disconnect Twitter at any time from the dashboard
- Revoke access by removing the Chrome Extension
- Request deletion of stored session data

Upon revocation, all related session data is immediately invalidated.

---

## 7. Policy Updates

We may update this policy periodically.

If a material change is introduced:
- A new policy version is published
- Users are required to re-accept the policy before continuing usage
